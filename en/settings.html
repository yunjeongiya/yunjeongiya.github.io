블로그 기능 가이드

댓글 기능

포스트 하단의 Terminal 패널에서 COMMENTS 탭을 클릭하면 GitHub 계정으로 댓글을 남길 수 있어요.

다국어 지원

포스트 우측 하단 상태바를 확인해보세요. 다른 언어 버전이 있으면 "한국어 → English" 같은 링크가 보여요.

목차 (Outline)

우측 패널의 OUTLINE에서 포스트의 제목 구조를 한눈에 보고 클릭하면 해당 섹션으로 바로 이동할 수 있어요.

미니맵 (Minimap)

우측 패널의 MINIMAP은 VSCode처럼 전체 글의 미리보기를 제공해요. 긴 글에서 현재 위치를 파악하기 좋아요.

Terminal 토글

좌측 하단 상태바의 Terminal을 클릭하면 태그와 댓글 패널을 숨기거나 보일 수 있어요.

태그

Terminal 패널의 TAGS 탭에서 포스트의 모든 태그를 확인할 수 있어요.
